# 💎 Pragueira Jewels

> Sitio web oficial de **Pragueira Jewels**, una marca de joyería artesanal inspirada en el mar y la costa gallega.

---

## 🌊 Sobre el proyecto

Este proyecto consiste en el desarrollo de un catálogo web para Pragueira Jewels, donde los usuarios pueden descubrir las diferentes colecciones y productos de la marca de una forma sencilla, elegante y adaptada a cualquier dispositivo.

Actualmente el catálogo permite visualizar las colecciones, explorar los productos y conocer la historia de la marca.

---

## ✨ Funcionalidades

- 📂 Catálogo organizado por colecciones.
- 💍 Visualización de productos.
- 🖼️ Ampliación de imágenes.
- 🔎 Filtrado de productos por colección.
- 📱 Diseño responsive.
- 📩 Contacto mediante Instagram y correo electrónico.
- 🚚 Información sobre cómo realizar pedidos.
